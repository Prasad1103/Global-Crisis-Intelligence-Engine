# Phase 1 — Project Planning

## Objective

Design a SQL portfolio project that demonstrates advanced PostgreSQL skills using real-world global datasets.

### Goals

- Build a PostgreSQL analytical database
- Practice advanced SQL concepts
- Generate business insights
- Create a portfolio-ready project

---

# Phase 2 — Dataset Collection

Five public datasets were selected.

| Dataset | Purpose |
|----------|---------|
| Peace Index | Measure global peace levels |
| Inflation | Analyze economic trends |
| Disasters | Measure disaster impact |
| Food Prices | Study commodity price trends |
| World Indicators | Analyze development metrics |

---

# Phase 3 — Data Understanding

Each dataset was explored to understand:

- Column names
- Data types
- Missing values
- Duplicate records
- Join opportunities
- Business meaning

---

# Phase 4 — Data Cleaning

Python and Pandas were used to prepare the datasets.

Tasks included:

- Removing unnecessary columns
- Standardizing column names
- Converting data types
- Cleaning missing values
- Formatting dates
- Exporting cleaned CSV files

---

# Phase 5 — Database Design

A PostgreSQL database was designed to store the cleaned datasets.

Database Name:

```text
global_crisis_intelligence
```

Tables Created:

- peace_index
- inflation
- disasters
- food_prices
- world_indicators

---

# Phase 6 — Data Loading

The cleaned CSV files were imported into PostgreSQL.

After loading, validation queries were executed to verify:

- Row counts
- Column types
- Data consistency
- Successful imports

---

# Phase 7 — SQL Analysis

SQL was used to explore and analyze the data.

Topics covered include:

- Filtering
- Sorting
- Aggregation
- Grouping
- Joins
- Subqueries

---

# Phase 8 — Advanced SQL

The project demonstrates several advanced PostgreSQL concepts.

Implemented features include:

- Window Functions
- Common Table Expressions (CTEs)
- Views
- User-defined Functions
- Indexes
- Query Optimization
- EXPLAIN ANALYZE

---

# Phase 9 — Business Reporting

Business-oriented SQL queries were developed to answer questions such as:

- Which countries have the highest inflation?
- Which countries experience the most disasters?
- Which commodities have the highest prices?
- How does peace relate to inflation?
- Which countries face multiple crisis indicators?
