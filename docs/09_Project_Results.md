# Project Results

## Project

**Global Crisis Intelligence Engine**

---

# Project Summary

The **Global Crisis Intelligence Engine** is an end-to-end SQL analytics project developed using PostgreSQL and Python.

The project integrates multiple global datasets into a unified analytical database, enabling cross-domain analysis of peace, inflation, disasters, food prices, and world development indicators.

The primary objective was to demonstrate advanced SQL skills while solving real-world analytical problems.

---

# Project Objectives Achieved

The following objectives were successfully completed:

- Designed a relational PostgreSQL database
- Cleaned and standardized multiple raw datasets
- Built an ETL pipeline using Python
- Loaded cleaned data into PostgreSQL
- Performed advanced SQL analysis
- Optimized query performance using indexes
- Created reusable SQL views and functions
- Implemented Common Table Expressions (CTEs)
- Used Window Functions for advanced analytics
- Generated executive-level business reports
- Documented the complete project
